# ガンバ大阪ニュースダッシュボード ⚽

Google NewsとX（Twitter）からガンバ大阪の最新情報を収集・表示するダッシュボードアプリです。

## 機能

- 🔍 キーワード検索
- 📰 Google Newsからのニュース取得
- 𝕏 X（Twitter）からの投稿取得
- 🎨 ガンバ大阪公式カラーのクールなデザイン

## 使用技術

- Python
- Streamlit
- feedparser

## ローカル実行

```bash
pip install -r requirements.txt
streamlit run ai_news_dashboard.py
```
